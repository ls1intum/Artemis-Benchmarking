Create Course and Exam

Creates a new course and a new exam with a few exercises.
Registers the necessary amount of test-users, generates student-exams and simulates the exam.
Note: The cleanup (= deletion of temporary courses and exams) is currently disabled for all modes!
On the Production instance, this mode requires ADMIN rights.

Existing Course and Unprepared Exam

Uses an existing course and exam. Please make sure that the necessary number of test-users is registered.
Student-exams will be (re-)generated. All existing student-exams will be deleted.
The start- and end-time of the exam will be changed. This change will NOT be undone!
Note that all registered students will be able to see and participate in the exam!
On the Production instance, this mode requires INSTRUCTOR rights in the corresponding course.

Existing Course and Prepared Exam

Uses an existing course and exam. Please make sure that the necessary number of test-users is registered.
Please make sure that the exam is fully prepared for conduction,
meaning that student-exams are generated and the conduction phase is active.
The exam will not be changed.

Existing Course, Create Exam

Uses an existing course and creates a new exam. Please make sure that the necessary number of test-users is registered for the course.
Registers the necessary amount of test-users to the exam, generates student-exams and simulates the exam.
The course will not be changed.
Note: The cleanup (= deletion of temporary courses and exams) is currently disabled for all modes!
On the Production instance, this mode requires INSTRUCTOR rights in the corresponding course.
